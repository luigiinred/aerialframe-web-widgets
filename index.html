AerialFrame Web Widgets

Beautiful, self-contained web widgets designed to be embedded in AerialFrame's WebView widget. Copy a URL, paste it in, and enjoy.

How to Use

1

Add a Web View Widget

In AerialFrame, go to Settings → Widgets → Add Widget → Web View.

2

Copy a Widget URL

Click "Copy URL" on any widget above. Customize it by adding URL parameters.

3

Paste the URL

In the Web View widget settings, paste the URL and optionally set an auto-refresh interval.

4

Customize

Each widget supports URL parameters for colors, content, and behavior. Check the README on GitHub for full docs.
